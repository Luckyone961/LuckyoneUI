### **About LuckyoneUI**  
• Active Discord community server for everything about WoW UIs (More than 5500 members).  
• All custom options mentioned below are disabled by default. You choose what you want to use.  
• All profiles are natively built on 1440p with clean custom scaling for 1080p users.  
• Everything is built specifically for high-end PvE content. Useless cosmetics disabled out of the box.  
• Free for everyone. No extra features locked behind subscriptions or one-time payments. Support is optional.  
• Full support for all official WoW versions such as Retail, Mists, Anniversary, Hardcore and Era.  
• Loads as a standalone Addon (For basic features) and ElvUI plugin (For the full experience).  
• Minimal performance footprint. Clean code, no dirty workaround fixes and frequent updates.  
• Provably the best performing setup for stable and high FPS. Countless reviews on Discord.  
• Updating the Addon will never auto-apply changes. Only if you manually re-run the ingame installer.  

### **Custom options**  
• Add a privacy overlay (streamer mode) on your guild community frame (click to show chat).  
• Automatically fill out the delete text when destroying items.  
• Build a clean custom chat with tabs in one click (Main, Log, Whisper, Guild, Party tabs).  
• Configure hidden Blizzard console variables in a user friendly options interface.  
• Disable annoying Blizzard features like Boss Banners and other screen center popups.  
• Toggles Blizzards cosmetic on-screen ffx effects and other visuals.  

### **Custom ElvUI features**  
• Aura filter setup for all classes and WoW versions.  
• Choose between two color themes (Dark default or class colored frames).  
• Layout presets (DPS, Tanks, Healers) for the best possible performance out of all UIs.  
• Lots of lightweight custom text format tags (Abbrev, Health, Name, Power, Targetname and more).  
• Raidframe handler for mythic raiders (Hides all benched people and only displays groups 1-4 in Mythic).  
• Small list of custom skins for third party Addons to match the general ElvUI style.

### **Ingame chat commands**  
• **/lucky config** [Opens the ingame options]  
• **/lucky install** [Run the installation process]  
• **/lucky minimap** [Hide or show the minimap icon]  
• **/lucky untrack** [Remove all tracked quests, including hidden, to boost performance]  
• **/luckydebug on** [Disables all Addons except ElvUI and LuckyoneUI for troubleshooting]  
• **/luckydebug off** [Re-Enables all Addons disabled by the command above]  
• **/vault** and **/weekly** [Open your great vault in retail to check weekly progress]
